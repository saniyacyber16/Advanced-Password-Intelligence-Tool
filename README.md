https://github.com/saniyacyber16/Advanced-Password-Intelligence-Tool/tree/main# Advanced Password Analyzer

Python GUI based cybersecurity tool that checks password strength and generates secure passwords.

## Features
- Password Strength Checker
- Entropy Calculation
- Estimated Crack Time
- Secure Password Generator
- Show / Hide Password
- Weak Password Detection

## Technologies Used
- Python
- Tkinter
- Regular Expressions
- Random module

## How to Run

git clone https://github.com/saniyacyber16/Advanced-Password-Intelligence-Tool

cd Advanced-Password-Intelligence-Tool

python advanced_password_tool.py
